TASK
1) Process each link in the links[] array.
2) FIRST: check if the link is an OC course link (openclassrooms.com/fr/ or /fr/courses/). If yes → MANUAL_REVIEW, add to manual_oc_links[], move to next link immediately.
3) For every non-OC link:
   a. Classify: type, pedagogical_purpose, authority_level.
   b. Read the html_context for this link — understand WHY this link is here and what the learner expects to find.
   c. Check the a4_decision_log — if A4 already swapped a related institution (e.g. CNIL→FTC), the replacement link must point to that swapped institution's site.
   d. Apply the replacement strategy for this type (government, wikipedia, technical doc, tool, article, etc.).
   e. Propose the best EN equivalent URL. Apply the quality gate — if it fails on any criterion, do not propose. Output NO_EQUIVALENT_FOUND with explanation.
   f. Assign confidence: HIGH, MEDIUM, or LOW.
4) Never invent URLs. Only propose URLs you are confident exist and are accessible.
5) Output the summary counts.
6) Output STRICT JSON only.
